    mysql使用指南

    初级 mysql sql
    创建时间:2020-02-05 12:37
    1. 增加
    2. 删除
    3. 修改

    show datebases; 显示应用中所有数据库
    use [yourdatabases]; 使用你需要的数据库
    show tables; 显示数据库中所有表

    show columns from 表名; 显示表中所有列

    增加

    create database 数据库名； 创建数据库
    create table 表名； 创建表
    insert into 表名 (filed1,field2…)values(value1,value2..); 表中插入数据

    删除

    drop table [yourtable]; 删除指定表
    drop database [yourdatabases]; 删除指定数据库
    delete from [yourtable] where field1 =’value1’; 删除表中符合where子句指定条件的数据

    修改

    update 表名 set field1=’value1’,field2=’value2’ where field1 =’value1’; 更新表中的符合查询条件的字段值
    alter table 表名1 +rename to 表名2 修改表名

    ​ +change 列名1 列名2 数据类型 将列名1修改为列名2

    ​ +drop 列名 删除表中列

    ​ +add 列名1 [数据类型（int，string）] after 列名2 在列2后增加表中列1

    ​ +modify 列名 数据类型2 修改列的数据类型。

    FIRST 和 AFTER 关键字可用于 ADD 与 MODIFY 子句
